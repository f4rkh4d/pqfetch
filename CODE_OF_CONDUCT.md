# code of conduct

this project follows the spirit of the
[contributor covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
condensed:

- be respectful in issues, PRs, discussions and review.
- assume good faith; no personal attacks; no harassment.
- no slurs, no doxxing, no off-topic political or sexual content.
- if someone behaves badly, email **hello@frkhd.com** with details and
  i'll handle it.

if you're new and unsure how to phrase something, that's fine, ask. if
you're experienced and tempted to be sharp, don't.
